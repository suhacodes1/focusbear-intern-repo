# Cyber Security

## Research & Learn

### 1. Common Cyber Security Threats in a Remote Work Environment

- Phishing attacks (emails or messages designed to steal information)
- Malware and malicious software downloads
- Weak or reused passwords
- Unsecured Wi-Fi networks
- Social engineering attacks
- Unauthorized access to devices or accounts

---

### 2. Best Practices for Securing Devices and Accounts

- Use strong, unique passwords for all accounts
- Enable two-factor authentication (2FA)
- Keep software and systems updated
- Avoid using public or unsecured Wi-Fi for sensitive work
- Install only trusted applications and tools
- Lock devices when not in use

---

### 3. Importance of Locking Your Computer

Locking your computer prevents unauthorized access to sensitive information. Even a short absence can expose data if the device is left unlocked, especially in shared or public environments.

---

### 4. Handling Phishing Attempts and Suspicious Links

- Do not click on unknown or suspicious links
- Verify the sender before responding to emails or messages
- Check for unusual requests or urgent language
- Report suspicious messages to the appropriate authority
- Avoid downloading attachments from unknown sources

---

### 5. Strong Passwords and Password Managers

A strong password should:

- Be long and complex
- Include a mix of letters, numbers, and symbols
- Be unique for each account

Using a password manager:

- Helps generate and store strong passwords securely
- Reduces the risk of password reuse
- Improves overall account security

---

### 6. Importance of Two-Factor Authentication (2FA)

Two-factor authentication adds an extra layer of security by requiring a second form of verification (e.g., a code sent to a device). It should be enabled on all important accounts, especially work-related tools and platforms.
