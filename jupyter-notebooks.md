# Using Jupyter Notebooks for Interactive Analysis

## Goal

Learn how to use Jupyter Notebooks for interactive data analysis and visualization.

---

## What I Did

I installed Jupyter Notebook using pip and created a new notebook file. I ran Python code inside cells to load and inspect a dataset using Pandas. I used functions like `head()`, `info()`, and basic filtering to explore the data. I also used Markdown cells to explain what each part of the code was doing and to document my observations.

---

## Reflection

### 1. What are the advantages of using Jupyter Notebooks for data analysis?

Jupyter Notebooks are useful because they combine code, output, and explanations in one place. This makes it easier to explore data step by step and immediately see results. It is also helpful for debugging since I can run one cell at a time instead of running an entire script.

---

### 2. How does Jupyter improve workflows compared to writing standalone Python scripts?

Jupyter improves workflows by allowing interactive execution. Instead of writing a full script and running it all at once, I can test small parts of the code, fix errors quickly, and iterate faster. It also helps keep track of my thought process since I can document each step directly in the notebook.

---

### 3. What are Markdown cells, and why are they useful in notebooks?

Markdown cells are used to write text, explanations, and notes inside the notebook. They are useful because they allow me to describe what the code is doing, explain results, and organise my analysis clearly. This makes the notebook easier to understand for both myself and others.

---

### 4. How could Jupyter Notebooks be used for analyzing Focus Bear’s user trends?

Jupyter Notebooks could be used to load user activity data from Focus Bear and analyse trends such as daily usage, feature engagement, or user retention. I could use Pandas to clean and process the data, and visualisation libraries like Matplotlib or Seaborn to create charts. Markdown cells would help explain the insights, making it easier to share findings with the team.

---

## Summary

In this task, I used Jupyter Notebooks to run Python code interactively, explore a dataset, and document my analysis using Markdown. This workflow makes data analysis more efficient, structured, and easier to communicate.
